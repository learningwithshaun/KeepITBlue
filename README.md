# KeepITBlue

**KeepITBlue** is a web platform for ocean and beach conservation, helping non-profits protect marine ecosystems through community engagement and environmental initiatives. 

---

## 📋 Overview

The platform provides NGOs with a **professional online presence** to:

* Share their mission and projects
* Coordinate volunteer efforts
* Facilitate donations
* Showcase organizational impact

Designed for **responsiveness and usability**, the website is accessible across devices and prioritizes a clear user experience.

---

## ✨ Features

* Multi-page website with consistent navigation and branding
* Donation system with payment options and form validation
* Volunteer recruitment showcasing opportunities
* Project showcase highlighting conservation initiatives
* Contact forms with client-side validation
* Responsive design for all devices
* Team profiles and organizational history

---

## 📁 Project Structure

```
├── index.html          # Homepage
├── about.html          # Organization history and team
├── contact.html        # Contact form
├── donate.html         # Donation page
├── projects.html       # Conservation projects
├── volunteer.html      # Volunteer opportunities
├── styles.css          # Global styling
├── script.js           # Main JavaScript functionality
├── formValidation.js   # Form handling and validation
└── images/             # Project assets
```

---

## 🛠️ Technologies Used

* **HTML5** – Page structure
* **CSS3** – Styling and layout
* **JavaScript (ES6)** – Dynamic behavior and form validation
* **Font Awesome Icons** – Visual enhancements
* **Google Fonts (Poppins)** – Typography
